# Getting Started with Python

Welcome to your Python learning journey! This guide will help you set up everything you need to start learning Python.

## Step 1: Install Python

### Windows
1. Go to [python.org](https://python.org)
2. Download Python 3.8 or higher
3. Run the installer
4. Check "Add Python to PATH"
5. Click "Install Now"

### macOS
```bash
# Using Homebrew
brew install python

# Or download from python.org
