# Python Variables & Data Types Cheat Sheet

## Basic Data Types

| Type | Description | Example |
|------|-------------|---------|
| `int` | Integer numbers | `42`, `-7`, `0` |
| `float` | Decimal numbers | `3.14`, `-2.5`, `0.0` |
| `str` | Text strings | `"hello"`, `'world'` |
| `bool` | Boolean values | `True`, `False` |
| `None` | Null value | `None` |

## Variable Assignment

```python
# Single assignment
name = "Alice"
age = 25

# Multiple assignment
x, y, z = 10, 20, 30

# Swap variables
a, b = b, a

# Constants (convention)
MAX_SIZE = 100
PI = 3.14159
