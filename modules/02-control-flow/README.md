## ğŸ“š What is Control Flow?

Control flow determines the order in which statements are executed. Without control flow, a program is just a sequence of instructions executed from top to bottom. With control flow, your program can:

- Skip code based on conditions.
- Execute code multiple times.
- Break out of loops early.
